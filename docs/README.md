# Документация фреймворка SpsFW

SpsFW - это PHP фреймворк, ориентированный на создание REST API. Он предоставляет функции маршрутизации, внедрения зависимостей (DI), валидации данных, управления доступом (авторизации) и другие инструменты, необходимые для разработки веб-сервисов.
